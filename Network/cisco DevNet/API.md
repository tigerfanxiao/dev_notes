# HTTP Methods

- GET 参数在 URL 里是明文的
- POST 参数在 body 里, 用来创建对象
- PUT 更新对象
- PATCH 只提高需要修改的部分属性
- DELETE 删除对象
### HTTP Response Code
- 1xx: Informational
- 2xx: Success
- 3xx: Redirection
- 4xx: Client Error
- 5xx: Server Error
