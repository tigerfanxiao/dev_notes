# 项目管理
概算和预算的区别
* 概算是还没有拿到图纸时候算的
* 预算是基于图纸，现实施工完了做的

1. 估算 那个表格就开始算
2. 概算 第一次设计完图纸
3. 预算 活干完了，修改施工图纸了，再算
4. 决算 结算是施工单位的做的， 是施工单位提交的
