# console
一般产品包装里有 console 先
DB9 公口(9 pin) + RJ45
COM口 也成 DB9 母口. 一般是电脑台式机的主板上有. 笔记本上需要 USB 转 COM, 并安装驱动
需要安装一个终端软件(xshell, secureCRT, putty).

### 结构
框式设备分为: 
* 主控版, 
* 接口板/业务版
* 服务版 一般用来扩容

一般业务板的数量和服务版成 2:1 的关系, 否则带不动
