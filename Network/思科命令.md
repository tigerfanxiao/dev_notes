
# 模式

```shell
enable # 进入可以 show 的模式
config ter # 进入全局配置模式
end # 退出所有配置层级
exit # 退出当前配置层级

```

# 保存配置
running-config 是在内存 cache中的配置, 如果没有 save 的话, 下一次重启的时候是会失效的
只有保存在startup-config中的配置, 在设备下一次重启的时候才能生效

```shell

wr # save
copy running-config startup-config
# 默认文件名

# startup-config 的保存路径是在 nvram 中的
dir nvram: 

reload # 重启设备
```
