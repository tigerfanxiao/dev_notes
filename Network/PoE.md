
最好用CAT6e 网线。 
网线布线从交换机到终端设备不能超过100米, 否则网速会衰减

# 部署方式
1. 直连PoE交换机和终端设备
2. 普通交换机(数据交换) +PoE Injector (供电) + 终端设备


### 协议
目前PoE有三种协议

| 协议    | 名字  | 功耗                  | 线缆       |
| ------- | ----- | --------------------- | ---------- |
| 802.3af | PoE   | 15.4 W 现在用的不多了 | CAT3和CAT5 |
| 802.3at | PoE+  | 30 W 现在用的比较多   | CAT5, 优选CAT6A       |
| 802.3bt(Type3) | PoE++ | 60 W                  |            |
| 802.3bt(Type4) | Higher-Power PoE  |  100W     |            |

# 常见设备功耗
摄像头功耗 一般不超过8W
AP一般功耗在20W
