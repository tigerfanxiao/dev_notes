# 算法

### Geohash算法
Hilbert Curve 将两维平面转为一维， 将经纬度转为整数。 整数越接近，经纬度越接近



2^64的整数， 可以表示宇宙中的任何一粒沙子


Peano Curve 将经纬度转为字符串， 前缀约接近，地理越接近

### Geo Fence
以城市为基础，将城市作为一个多边形。 如何求一个点在一个对边型内。
如果人在两个城市之间，可以同时对两个城市做查询
