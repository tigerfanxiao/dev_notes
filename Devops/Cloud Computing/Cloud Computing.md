云的计算资源类型可以分为 4 种
- Shared 类似 VM, 公用一个物理机上, 有多个 VM
- Transient 类似 Spot VM
- Reserved 
- Dedicated 类似 Bare Metal 
# Virtual Machine
## Hypervisor Types
有两种 hypervisor
一种是用 type1, 类似 exsi, 是直接把 hypervisor 装在裸金属设备(Bare Metal)上的, 没有操作系统. 另一种是 type2, 类似 VMware workstation. 先装了操作系统, windows, 再装hypervisor
