# 模式
vim 有下面 4 中模式
* normal
* Insert
* visual
* 命令模式

## 模式转换
### Normal -> Insert
在光标前插入 i
在行首插入 I

在光标后插入 a
在行尾插入 A

在下一行插入 o
在上一行插入 O
