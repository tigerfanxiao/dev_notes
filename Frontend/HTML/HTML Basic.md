# Form
## Input
### Input Type

#### Input text
```html
<input type="text" 
	   placeholder="this is default text"/>
```
#### Input Number 
有箭头可以控制数字
```html
<input type="number" />
```
#### Input Range
```html
<input type="range" min="0" max="100"/>
```

#### Checkbox
```html
<input type="checkbox"/> Purple
```
### Input Event
- onChange
- onFocus
- onBlur 当光标移开输入框
## Drop Menu
```html
<select>
	<option value="role">role</option> <!-- 一般是放一个 role 作为默认值 -->
	<option value="Engineer">Engineer</option>
	<option value="Teacher">Teacher</option>
</select>
```
