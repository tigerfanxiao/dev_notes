# windows
alt + F12 打开 terminal
esc 回到代码编辑窗口


# mac
### 代码编辑
option + up 选中整个单词
ctrl + G 选中重复出现的单词, 每按一次 G 会向下查找
option + delete 删除一个单词
cmd + delete 删除完整一行

### 窗口控制
ctrl + tab 在打开的标签页中切换


### 代码跳转
cmd + b 跳转到变量定义的地方
